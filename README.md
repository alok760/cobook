# cobook

## Installation

- Install pipenv - https://pypi.org/project/pipenv/
- Clone the repository - `git clone https://github.com/alok760/cobook.git`
- `cd cobook`
- `pipenv shell`
- `pipenv install`
- `python manage.py migrate`
- `python manage.py runserver`
